# Football Ticket Booking System

A PostgreSQL-based relational database project focused on designing a structured football ticket booking system using database normalization, ERD modeling, and SQL queries.

## Tech Stack
- PostgreSQL
- SQL

## Features
- Designed a normalized database schema with three tables: Users, Matches, and Bookings
- Created an ERD using primary keys, foreign keys, and Crow's Foot notation
- Implemented 7 SQL queries demonstrating filtering, pattern matching, NULL handling, JOIN operations, subqueries, and pagination

## SQL Queries
1. Retrieve data using category and status filters
2. Search records with `LIKE` and `ILIKE`
3. Handle NULL values using `COALESCE`
4. Fetch related data using `INNER JOIN`
5. Include unmatched records with `LEFT JOIN`
6. Use a subquery with `AVG` aggregation
7. Implement pagination using `LIMIT` and `OFFSET`
